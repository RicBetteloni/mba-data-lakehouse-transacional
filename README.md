# Data Lakehouse Transacional (Tema A)

Projeto final da disciplina Data Collection & Storage.

## Tecnologias

- Python / PySpark
- Delta Lake

## Estrutura do notebook

1. Configuração do Spark + Delta
2. Coleta/Aquisição (camada RAW em Parquet)
3. Pré-processamento (3 transformações de qualidade)
4. Camada CURATED em Delta Lake
5. Análises de negócio (KPIs + comparativo Parquet x Delta)
6. Operações ACID (INSERT, DELETE, UPSERT) + Time Travel
7. Resumo do pipeline

## Como executar

1. Abra o notebook no Google Colab.
2. Execute as células em ordem, de cima para baixo.
3. Ao final, verifique as mensagens de sucesso e os KPIs/operacoes ACID.
